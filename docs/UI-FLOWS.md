# UI/UX Flows

## Design Principles

1. **Mobile-first**: Teacher app designed for low-end Android phones
2. **Large touch targets**: Minimum 44px height for all interactive elements
3. **Fast feedback**: Optimistic updates with sync indicators
4. **Minimal typing**: Dropdowns, toggles, and pre-filled defaults
5. **Offline-aware**: Clear status indicators when offline

### Attendance Toggle (Touch)

2. Tap status button to change (P â†’ A â†’ L â†’ E â†’ S â†’ P)
3. Long press for quick notes

### Form Validation

- Real-time validation with error messages
- Required fields marked with *
- Submit button disabled until valid
- Toast notification on save

### Loading States

- Skeleton loaders for lists
- Spinner for buttons during action
- Pull-to-refresh on mobile

### Error Handling

- Toast notifications for errors
- Retry buttons for failed actions
- Clear error messages

- Yellow banner when offline
- Actions still work (queued locally)
- Sync indicator shows pending count
- Auto-sync when online
